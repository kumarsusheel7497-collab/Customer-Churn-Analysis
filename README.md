Customer Churn Analysis

Overview

This project focuses on analyzing customer data to understand the reasons behind subscription cancellations and customer churn. The analysis helps identify patterns in customer behavior, engagement, contract type, payment methods, tenure, and charges that influence whether a customer leaves or stays.

Objective

The main objective of this project is to analyze customer churn data, identify the factors that contribute to churn, and provide insights that can help improve customer retention.

Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

Steps Performed

- Loaded the customer churn dataset
- Inspected rows, columns, missing values, and duplicates
- Removed duplicate rows
- Cleaned and standardized column names
- Handled missing values
- Converted incorrect data types
- Analyzed churn distribution
- Visualized churn by gender, contract type, and payment method
- Examined monthly charges and tenure distributions
- Generated a correlation heatmap for numeric features
- Saved the cleaned dataset for further analysis

Files in this Repository

- "Customer_Churn_Analysis.ipynb" → Colab notebook containing the full analysis
- "cleaned_customer_churn.csv" → Cleaned churn dataset
- "README.md" → Project documentation

Key Insights

The analysis helps identify customer groups with higher churn rates based on subscription type, charges, and service usage patterns. These insights can help businesses improve retention strategies and reduce customer loss.
